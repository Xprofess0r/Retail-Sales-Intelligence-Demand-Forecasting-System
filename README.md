# 🛒 Retail Sales Intelligence & Demand Forecasting System

## 🔗 Project Overview

This project focuses on analyzing retail sales data to uncover business insights and forecast future demand. It combines **SQL, Python, and Power BI** to build an end-to-end analytics pipeline that supports data-driven decision-making for inventory management, revenue optimization, and sales strategy.

---

## 🎯 Objectives

* Analyze sales trends and product performance
* Identify high-revenue categories and seasonal demand patterns
* Build a forecasting model to predict future sales
* Create an interactive dashboard for business stakeholders

---

## 🛠 Tech Stack

* **SQL** – Data storage, querying, and aggregation
* **Python (Pandas, NumPy, Scikit-learn)** – Data cleaning, feature engineering, and machine learning
* **Power BI** – Interactive dashboard and visualization
* **Excel / CSV** – Data source

---

## 🏗 Project Architecture

Raw Data → SQL Database → Python Analysis & ML → Power BI Dashboard

---

## 📊 Key Features

### 🔹 Data Analysis (SQL + Python)

* Revenue analysis by category and time
* Top-selling products identification
* Monthly and seasonal sales trends
* Customer and product-level insights

---

### 🔹 Demand Forecasting (Machine Learning)

* Feature engineering using time-based variables and lag features
* Built a forecasting model to predict future sales
* Evaluated model performance for accuracy and reliability

---

### 🔹 Power BI Dashboard

* KPI cards (Total Revenue, Sales Growth, Top Products)
* Time-series sales trends
* Category-wise performance
* Interactive filters (date, product category)

---

## 📈 Key Insights

* Sales show strong seasonal patterns across months
* A small set of products contributes to a large portion of revenue
* Demand peaks during specific time periods, indicating inventory planning needs
* Forecasting helps anticipate demand fluctuations and optimize stock levels

---

## 🚀 Business Impact

* Enables better inventory planning
* Helps reduce stockouts and overstocking
* Supports data-driven pricing and promotion strategies
